# Smart Task Reminder

A Smart Task Reminder application built using:

- React.js (Frontend)
- Flask (Backend)
- Supabase (Database)

## Features
- Add Tasks
- View Tasks
- Mark Tasks as Completed
- Delete Tasks
- Store Data in Supabase

## Technologies Used
- React
- Flask
- Python
- Supabase
- GitHub
